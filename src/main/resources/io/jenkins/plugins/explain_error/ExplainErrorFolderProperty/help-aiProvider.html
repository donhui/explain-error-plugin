Select and configure the AI provider to use for error explanation in this folder.

Supported Providers:
  • OpenAI: Requires API key and model selection (e.g., gpt-4)
  • Google Gemini: Requires API key and model selection (e.g., gemini-2.5-flash)
  • Ollama: Requires URL to Ollama instance and model name

If not configured here, the global configuration will be used as fallback.
